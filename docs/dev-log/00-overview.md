# 사이드 프로젝트 평가기 - 개발 이력

## 프로젝트 개요
사람들이 사이드 프로젝트 개념(텍스트)이나 GitHub 링크를 제출하면,
6가지 관점에서 자동 분석하여 100점 만점 평가 리포트를 생성하는 웹 서비스.

## 평가 영역 (100점 만점)
| 영역 | 배점 | 설명 |
|------|------|------|
| 기술 완성도 | 20점 | 스택, 아키텍처, 코드 품질, 확장성 |
| 시장 적합성 | 20점 | 시장 규모, 경쟁, 차별화, PMF |
| 사용자 경험 | 15점 | UX/UI, 접근성, 온보딩 |
| 실현 가능성 | 15점 | 난이도, 리소스, 일정, 유지보수 |
| 성장 잠재력 | 15점 | 수익 모델, 마케팅, 성장 전략 |
| 리스크 관리 | 15점 | 기술/법적/시장/의존성 리스크 |

## 기술 스택
- **Frontend**: Next.js 15, React 19, Tailwind CSS, TypeScript
- **Backend**: Next.js API Routes, Prisma ORM
- **DB**: SQLite (로컬) → PostgreSQL (프로덕션)
- **배포**: Vercel (예정)
- **개발 도구**: Claude Code Team Mode (5 workers 병렬)
