# Side Project Evaluator - Agent Coordination

## 팀 구성

이 프로젝트는 6개의 전문 평가 에이전트로 구성됩니다.
각 에이전트는 독립적으로 분석을 수행하며, 오케스트레이터가 결과를 종합합니다.

## 에이전트 목록

### 분석 에이전트 (병렬 실행)
- `tech-analyst`: 기술 스택, 아키텍처, 코드 품질, 확장성 (20점)
- `market-analyst`: 시장 규모, 경쟁, 차별화, PMF (20점)
- `ux-reviewer`: UX/UI, 접근성, 온보딩 (15점)
- `feasibility-checker`: 개발 난이도, 리소스, 일정 (15점)
- `growth-strategist`: 수익 모델, 마케팅, 성장 전략 (15점)
- `risk-assessor`: 기술/법적/시장 리스크 (15점)

## 실행 규칙

1. **병렬 실행**: 6개 에이전트는 반드시 동시에 실행
2. **독립성**: 각 에이전트는 다른 에이전트의 결과에 의존하지 않음
3. **출력 형식**: 각 에이전트는 정해진 마크다운 형식으로 출력
4. **언어**: 모든 출력은 한국어
5. **객관성**: 근거 없는 추측 금지, 분석 근거를 명시

## 점수 체계

총점 100점:
- 기술 완성도: 20점 (4개 세부 항목 x 5점)
- 시장 적합성: 20점 (4개 세부 항목 x 5점)
- 사용자 경험: 15점 (4개 세부 항목)
- 실현 가능성: 15점 (4개 세부 항목)
- 성장 잠재력: 15점 (3개 세부 항목 x 5점)
- 리스크 관리: 15점 (4개 세부 항목)
